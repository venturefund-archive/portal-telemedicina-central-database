## Project Setup

To set up the frontend project, follow these steps:

1. Install dependencies:
   ```sh
   npm install
   ```

2. Compile and hot-reload for development:
   ```sh
   npm run dev
   ```

3. Compile and minify for production:
   ```sh
   npm run build
   ```
