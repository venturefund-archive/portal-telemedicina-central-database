Child development indicators¶

Here we present a proposal for initial indicators relevant for monitoring child development.

The Indicator column refers to the high-level concepts that will be monitored by the system. The table below relates the indicators to typical health-related data sources. The data required for the calculation of the indicators are mapped to HL7 FHIR Resources to ensure a minimal interoperability between health systems.
